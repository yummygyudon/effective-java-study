# [Item25] 톱 레벨 클래스는 한 파일에 하나만 담으라


<br/>

---
#### 🌟 핵심 🌟
```text
* "소스 파일 하나"에는 반드시 "톱레벨 클래스/인터페이스 하나만" 담자.
  - 컴파일러가 한 클래스에 대한 정의를 여러개 만드는 일이 사라질 것이다.

* 소스 파일을 어떤 순서로 컴파일하든 
  바이너리 파일이나 프로그램의 동작이 달라지는 일은 결코 일어나지 않을것이다.

* 단순히 톱레벨 클래스들은 서로 다른 소스 파일로 분리하자.
```

<br/>
